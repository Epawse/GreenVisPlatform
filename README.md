# 城市绿色空间展示平台

基于 Vue 3 + OpenLayers + Vite 构建的城市绿地可达性与公平性可视化分析系统

## 环境要求

- Node.js >= 16.0.0
- pnpm >= 8.0.0

## 安装依赖

**本项目使用 pnpm 作为包管理器**

```bash
pnpm install
```

## 开发

```bash
pnpm dev
# 或
pnpm serve
```

开发服务器将在 http://localhost:8080/ 启动

## 构建

```bash
pnpm build
```

## 预览构建结果

```bash
pnpm preview
```

## 代码格式化

```bash
# 格式化所有代码
pnpm format

# 检查代码格式
pnpm format:check
```

## 部署

```bash
pnpm deploy

# 或者构建并部署
pnpm build-and-deploy
```
